# Настройка JSONBin.io для отзывов

## Шаг 1: Создание аккаунта (опционально)

1. Перейдите на https://jsonbin.io/
2. Нажмите "Sign Up" (можно использовать бесплатный план)
3. Или используйте без регистрации (публичные bin'ы)

## Шаг 2: Создание Bin

### Вариант A: С регистрацией (рекомендуется)

1. После регистрации нажмите "Create Bin"
2. Вставьте начальный JSON (объект с массивом reviews):
```json
{"reviews": []}
```
3. Нажмите "Create"
4. Скопируйте **Bin ID** (например: `507f1f77bcf86cd799439011`)
5. Скопируйте **Access Key** из настроек (Settings → Access Keys)

### Вариант B: Без регистрации (публичный bin)

1. Перейдите на https://jsonbin.io/
2. Вставьте начальный JSON (объект с массивом reviews):
```json
{"reviews": []}
```
3. Нажмите "Create"
4. Скопируйте **Bin ID** из URL
5. API Key не требуется для публичных bin'ов

## Шаг 3: Настройка в коде

Откройте файл `src/App.js` и найдите строки:

```javascript
const JSONBIN_ID = 'YOUR_BIN_ID'; // Замените на ваш bin ID
const JSONBIN_API_KEY = 'YOUR_API_KEY'; // Замените на ваш API ключ
```

### Если используете публичный bin (без регистрации):

```javascript
const JSONBIN_ID = 'ваш_bin_id';
const JSONBIN_API_KEY = '$2b$10$ваш_публичный_ключ'; // Публичный ключ из bin
```

### Если используете с регистрацией:

```javascript
const JSONBIN_ID = 'ваш_bin_id';
const JSONBIN_API_KEY = 'ваш_мастер_ключ'; // Master Key из настроек
```

## Шаг 4: Проверка

1. Запустите проект: `npm start`
2. Перейдите в секцию "Отзывы"
3. Попробуйте оставить отзыв
4. Проверьте, что отзыв появился в списке

## Важные замечания

- **Бесплатный план JSONBin.io:** 1000 запросов/день
- **Публичные bin'ы:** доступны всем, кто знает ID
- **Приватные bin'ы:** требуют API ключ для доступа
- **Модерация:** можно удалять отзывы через веб-интерфейс JSONBin.io

## Альтернативные варианты

Если JSONBin.io не подходит, можно использовать:
- Firebase Realtime Database (бесплатный план)
- Supabase (бесплатный план)
- Google Sheets + Apps Script
